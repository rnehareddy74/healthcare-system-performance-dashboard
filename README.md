# healthcare-system-performance-dashboard
Executive-level healthcare analytics dashboard visualizing patient volume, financial performance, satisfaction, and 30-day readmission trends across a regional hospital network using Power BI.
This project delivers a comprehensive executive dashboard that integrates clinical, operational, and financial healthcare data to provide leadership with actionable insights. Key features include:

Top KPIs: Total patients, total revenue, average patient satisfaction, 30-day readmission rate

Operational Insights: Visit volume analysis by visit type and hospital

Financial Analysis: Revenue by hospital, insurance type breakdown, and visit type segmentation

Quality Metrics: Monthly readmission trends by visit type

Interactive Filtering: Hospital-level slicers for focused analysis

Tools Used: Power BI, DAX, Data Modeling,
